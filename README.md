# FLASH

FLASH is a companion-style workspace project.

## Current milestone

SP1 Bootstrap v0.1 establishes the minimum engineering foundation:

- application entrypoint
- centralized paths
- JSON configuration
- service registry
- event bus
- logger service
- test package
- plugin and workspace boundaries

## Run

```powershell
python main.py
```

Expected output:

```text
FLASH SP1 bootstrap running.
```

## Test

```powershell
python -m pytest
```
